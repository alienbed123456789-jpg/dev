# TimeFlow - Activity & Attendance Tracker

A modern, full-stack web application designed for tracking daily activities, work hours, vacations, and holidays. The project is structured as a monorepo containing both the Frontend (React) and Backend (Spring Boot).

## Quick Start (One-Command Run)

To spin up the entire application from a fresh clone, use the provided start scripts located in the root directory:

**For Windows:**
Simply double-click the `start.bat` file.

**For macOS/Linux/Git Bash:**
Open your terminal and run:
```bash
chmod +x start.sh
./start.sh
